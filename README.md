iOSAlipayUI
===================================  
  提示  
-----------------------------------  
  本app主要演示了：
  动态显示splashimage
  
  如何控制手势密码（AES加密保存数据）
  
  如何控制viewcontroller之间的跳转，viewcontroller的代理控制。
  
  TabbarController与navigationControlle结合使用
  
  动态显示splashimage
  
  定制tableViewController
  
  自定义UIButton
